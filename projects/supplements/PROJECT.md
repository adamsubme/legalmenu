## Description

VitaBites to marka suplementów diety (gummies / żelki witaminowe). VitalCore to ramię produkcyjne — kontraktowa produkcja suplementów diety. Model biznesowy: B2B kontraktowa produkcja z brandingiem VitaBites.

---

## Business Model

### Oferta
- **Kontraktowa produkcja suplementów diety** (gummies, kapsułki, tabletki)
- **Private label** — klient dostaje własną markę
- **Co-branding** — VitaBites + marka klienta

### Target Personas
- **A:** Startup Health — młode firmy szukające producenta
- **B:** E-commerce Supplements — sprzedawcy online
- **C:** Traditional Pharma — tradycyjne firmy farmaceutyczne
- **D:** Private Label — sieci handlowe, apteki

### Sales Tracks
- **Express:** Szybki pilot (1-2 tyg), minimalne zamówienie
- **Pilot:** Test production run (2-4 tyg)
- **Custom:** Pełny kontrakt (indywidualny)
